# 問題文

### 書籍名と総ページ数の一覧を取得してください。出力項目は書籍名(name)と総ページ数(total_page)です。※ AS 句を使用して表示するカラム名を変更してください。

※ DB: PostgreSQL ver14.4

# SQL

```sql
SELECT name AS 書籍名, total_page AS 総ページ数 FROM books
```
