# 問題文

### 書籍の税込価格一覧を取得してください。消費税率は 10%です。出力項目は book_id(書籍 ID)と tax_included_price(税込価格)

※ DB: PostgreSQL ver14.4

# SQL

```sql
SELECT  book_id, price * 1.1  AS tax_included_price
from book_sales
```
